#Room Allocation Module

This module helps you to allocate people(fellows and staff) to rooms from a text file stored in the data folder.

####Running the module off-line
  1. Check for whether python is installed on your computer, if not you are required to install python from this [link](https://www.python.org/).
  
  2. Install nose  by typing this command
  
  ```
  sudo apt-get install python-nose
 ```
  
  2. Initialise a git folder
    ``` 
    git init 
    ```
  3. Clone the github repo using the command
    ```
    git clone git@github.com:andela-engmkwalusimbi/checkpoints.git
    ```
  4. To run the module navigate to where you cloned the repo and type the following commands in terminal
  ```
  python entry.py
  ``` 
  press enter
  
  5.  To run tests for the module run the following command
  ```
  nosetests test/test_checkpoint1.py
  ```
